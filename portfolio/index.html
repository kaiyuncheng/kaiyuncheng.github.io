Articles Website

Overview

Capture Studio is an e-commerce website using Vue.js. I started this side project from scratch, which involved designing the layout, creating wireframes, and programming. The website features include showcasing photography services/products, client albums, a shopping cart, and a checkout system. On the backend platform, functionalities include login authentication, management of services/products for listing and publishing, coupon management, order management, and image management.

Technologies Used:

  • Vue.js / Vue Router / Vue Cli
  • ESLint with the Airbnb syntax standards
  • RESTful API
  • Bootstrap
  • Responsive Web Design
  • SCSS
  • Git Version Control
